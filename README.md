# post2pages — eBay to Facebook Automation Engine

A high-efficiency automation tool designed to bridge the gap between e-commerce inventory and social media marketing. This system monitors eBay store activity and automatically generates formatted promotional posts for Facebook Pages.

## 🚀 Overview
Managing multi-channel inventory requires speed and consistency. **post2pages** eliminates the manual overhead of cross-posting by utilizing the eBay API and Facebook Graph API to synchronize listings in real-time.

## 🛠 Key Features
* **Automated Sync:** Pulls active listings from the `esquireattire` and `fixboatquick` eBay stores.
* **Smart Formatting:** Converts eBay item descriptions and pricing into high-conversion Facebook post layouts.
* **Image Processing:** Handles high-resolution image variants to ensure visual quality on social feeds.
* **Affiliate Integration:** Automatically injects custom affiliate URL formatting for revenue tracking.
* **Headless Execution:** Powered by GitHub Actions and PHP for 24/7 reliability without a dedicated local server.

## 🏗 Technology Stack
* **Language:** PHP 8.x
* **Automation:** GitHub Actions (CI/CD)
* **APIs:** eBay Developer API, Facebook Graph API
* **Data Handling:** JSON-based inventory tracking
